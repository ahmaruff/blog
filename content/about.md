Seorang software developer dengan fokus pada Web Development. Halaman ini saya gunakan untuk berbagi catatan pribadi tentang apa yang sedang saya pelajari,
proyek yang saya kerjakan, dan pengalaman sehari-hari di dunia teknologi.

Blog ini adalah ruang dokumentasi bagi saya sendiri, di mana setiap tulisan mencerminkan proses belajar, tantangan yang ditemui, dan solusi yang ditemukan selama perjalanan.

---

### Perjalanan Profesional

Saya mulai mengeksplorasi dunia IT sejak masa sekolah dan kuliah, dari administrasi jaringan, web development, hingga pemahaman dasar tentang AI/ML.

Dalam perjalanan karir, saya banyak bekerja dengan web fullstack, terutama menggunakan PHP (Laravel), dan terlibat dalam pengembangan sistem internal, platform e-commerce,
serta integrasi dengan layanan pihak ketiga seperti WhatsApp Cloud dan OpenAI.

Pengalaman ini memberi saya kesempatan memahami bagaimana membangun sistem yang sederhana, efisien, dan mudah dipelihara, sambil terus belajar meningkatkan kualitas kode dan desain arsitektur.

### Di Luar Pekerjaan

Di luar pekerjaan, saya biasanya menghabiskan waktu dengan membaca buku non-fiksi maupun novel, untuk mengeksplor ide baru dan memperluas perspektif.
Selain itu, saya juga menikmati kegiatan di luar rumah, seperti mendaki gunung, berolahraga, atau sekadar berjalan santai untuk melepas penat dari layar monitor.
